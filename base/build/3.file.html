# 3.MongoDB 启动与连接

# 3.1windows 启动服务器端

  • 1.找到 mongodb 安装目录，一般是C:\Program Files\MongoDB 2.6 Standard\bin
  • 2.按下 Shift+鼠标右键,选择在此处打开命令窗口
  • 3.在除 C 盘外的盘符新建一个空目录,如 D:\Mongodb\data
  • 4.在命令行中输入 mongod --dbpath=刚创建的空目录,如
mongod --dbpath=D:\Mongodb\data
1
  • 5.再按回车键 如果出现 waiting for connections on port 27017 就表示启动成功,已经在 27017 端口上监听了客户端的请求注意：--dbpath 后的值表示数据库文件的存储路径,而且后面的路径必须事先创建好，必须已经存在，否则服务开启失败注意：这个命令窗体绝对不能关,关闭这个窗口就相当于停止了 mongodb 服务
