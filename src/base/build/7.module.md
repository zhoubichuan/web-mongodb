# 六.集合操作

### 6.1 查看集合帮助

```sh
db.students.help()
```

### 6.2 查看数据库下的集合

```sh
show collections
```

### 6.2 创建集合

#### 6.3.1 创建一个空集合

```sh
db.createCollection(collection_Name)
```

- collection_Name 集合的名称

### 6.3.2 创建集合并插入一个文档

- collection_Name 集合的名称
- document 要插入的文档

```sh
db.collection_Name.insert(document)
```
